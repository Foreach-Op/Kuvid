Shooter Rotate-> Q,E

Transform(if Blend mode is activated): (1,2,3,4) then (1,2,3,4) 
Pick Powerup -> (alt)+(1,2,3,4)
Pause -> P
Resume -> R
Save -> S (not functioning now)
